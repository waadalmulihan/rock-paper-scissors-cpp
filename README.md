# 🎮 Rock Paper Scissors Game - C++

## 📌 Project Description

This is a console-based Rock Paper Scissors game developed in C++.
The player competes against the computer over multiple rounds.

---

## ✨ Features

* Player vs Computer gameplay
* Random computer choices
* Multiple rounds (1–10)
* Score tracking system
* Final winner calculation
* Replay option
* Screen reset between games

---

## 🎮 How to Play

1. Run the program
2. Enter number of rounds (1–10)
3. Choose:

   * 1 = Stone
   * 2 = Paper
   * 3 = Scissors
4. Play against the computer
5. View round results
6. See final winner

---

## 📊 Game Logic

* Stone beats Scissors
* Scissors beats Paper
* Paper beats Stone
* Same choice = Draw
